These are preliminary notes relative to this release of beat.
SVN repository: https://beat.sourceforge.net/svnroot/beat/trunk

Libraries required (to compile):

Jena (jena.sourceforge.net) version used in this distribution: 2.5.7 (testing: 2.6.0)
Pellet (clarkparisa.com/pellet). Version used in this distribution: 2.0.0 rc4
	- pellet-core.jar
	- pellet-jena.jar
Secondstring (secondstring.sourceforge.net). Version used in this distribution: 15/6/2006
UEA Stemmer  (http://fizz.cmp.uea.ac.uk/Research/stemmer/)
Alignment APIs (https://gforge.inria.fr/projects/alignapi/). Version used in this distribution: 3.5 (testing: 3.6)
	- procalign.jar
ANTLR runtime libraries (http://www.antlr.org/). Version used in this distribution: 3.1.1
MySQL jdbc driver (http://dev.mysql.com/downloads/connector/j/3.1.html)
